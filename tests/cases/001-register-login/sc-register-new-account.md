## TC-007: 注册成功——创建账户、自动登录并重定向

- **优先级**: P1
- **前置条件**:
  - 数据库中无目标用户名的已有账户
  - 用户未持有有效 JWT Cookie

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 填写合法用户名（如 `newuser01`）+ 合法密码（`Test@1234`）+ 一致确认密码 + 合法邮箱（`newuser01@example.com`），点击"注册" | 表单提交发送 |
| 2 | 观察"注册"按钮状态 | 按钮进入加载/禁用状态（防止重复提交），显示 loading 指示器 |
| 3 | 注册完成后查看浏览器 HTTP 响应 | HTTP 302 状态码，`Set-Cookie` 响应头中包含 `HttpOnly` 标记的 JWT Token |
| 4 | 查看浏览器存储的 Cookie | JWT Cookie 已设置，属性包含 `HttpOnly`，`SameSite` 值为 `Lax` 或 `Strict` |
| 5 | 查看当前页面 URL | 已重定向至 `/console`，状态码为 200 |
| 6 | 数据库查询用户名 `newuser01` | 账户已创建，状态为活跃，创建时间为当前时间附近 |

**验证方式**: 集成 + E2E
**自动化文件**: `tests/integration/001-register-login/test_register.py`, `tests/e2e/001-register-login/test_register_login_flow.spec.ts`
